## Upload to Overleaf

1. Go to [overleaf.com](https://www.overleaf.com) → **New Project** → **Upload Project**
2. Upload **`presentation-overleaf.zip`** (from parent folder)
3. Main document: **`main.tex`**
4. **Recompile** → **Download PDF**
